# EvolveHer – A Personal Wellness & Period Tracker.

**EvolveHer** is a simple Java application designed to help users track their menstrual cycles, monitor cycle phases, and maintain personal journals — all in one wellness companion.

---

## Central Idea

A complete cycle and self-care tracker where users can:
- Log their period details.
- View calculated cycle summaries.
- Check which phase they’re currently in.
- Write and view journals for emotional tracking.
- Receive lifestyle and self-care suggestions for each phase.

## Project Structure


```markdown
EvolveHer/
│
├── EvolveHer.java         # Main menu and navigation  
├── PeriodTracker.java     # Handles cycle input and summary  
├── CyclePhase.java        # Displays phase insights and tips  
└── JournalTracker.java    # Journal writing and viewing  

```

##How to Run

Ensure you have Java 17+ installed.

Place all .java files in one folder.

Compile and run:

javac *.java
java EvolveHer
